# EH5160-stepprt

EH5160-stepper是用于基于tmc5160电机驱动器的 2 相电机的步进电机驱动器板。

目前模块还在测试中仅先放出原理图，gerber文件和bom表。只建议在设计时可以参考，目前元件还有更换的可能。不是很建议打板焊接。
